\section*{Draw}

Function \verb$draw(f,x)$ draws a graph of $f$ of $x$.
The second argument can be omitted in which case literal \verb$x$ is the default symbol.

{\color{blue}
\begin{verbatim}
draw(x^2)
\end{verbatim}}

% -2,2        2,2
% *-------------*
% |             |
% |             |
% |             |
% |             |
% |             |
% *-------------*
% -2,-2       2,-2

\begin{center}
\begin{tikzpicture}
\draw (-2,-2) -- (2,-2);
\draw (-2,0) -- (2,0);
\draw (-2,2) -- (2,2);
\draw (-2,-2) -- (-2,2);
\draw (0,-2) -- (0,2);
\draw (2,-2) -- (2,2);
\draw (-2,-2.3) node {\scriptsize$-10$};
\draw (2,-2.3) node {\scriptsize$10$};
\draw (-2.4,2) node {\scriptsize$\phantom+10$};
\draw (-2.4,-2) node {\scriptsize$-10$};
\draw [ultra thick] (-.6325,2) parabola bend (0,0) (.6325,2);
\end{tikzpicture}
\end{center}

Symbols \verb$xrange$ and \verb$yrange$ control the scale of the graph.

{\color{blue}
\begin{verbatim}
xrange = (-1,1)
yrange = (0,2)
draw(x^2)
\end{verbatim}}

\begin{center}
\begin{tikzpicture}
\draw (-2,-2) -- (2,-2);
%\draw (-2,0) -- (2,0);
\draw (-2,2) -- (2,2);
\draw (-2,-2) -- (-2,2);
\draw (0,-2) -- (0,2);
\draw (2,-2) -- (2,2);
\draw (-2,-2.3) node {\scriptsize$-1$};
\draw (2,-2.3) node {\scriptsize$1$};
\draw (-2.4,2) node {\scriptsize$\phantom{+0}2$};
\draw (-2.4,-2) node {\scriptsize$\phantom{-0}0$};
\draw [ultra thick] (-2,0) parabola bend (0,-2) (2,0);
\end{tikzpicture}
\end{center}

\newpage

Parametric drawing occurs when $f$ is a vector.
Symbol \verb$trange$ controls the parameter range.
The default is \verb$trange=(-pi,pi)$.
In the following example, \verb$draw$ varies \verb$theta$
over the default range $-\pi$ to $+\pi$.

{\color{blue}
\begin{verbatim}
xrange = (-10,10)
yrange = (-10,10)
f = 5 (cos(theta),sin(theta))
draw(f,theta)
\end{verbatim}}

\begin{center}
\begin{tikzpicture}
\draw (-2,-2) -- (2,-2);
\draw (-2,0) -- (2,0);
\draw (-2,2) -- (2,2);
\draw (-2,-2) -- (-2,2);
\draw (0,-2) -- (0,2);
\draw (2,-2) -- (2,2);
\draw (-2,-2.3) node {\scriptsize$-10$};
\draw (2,-2.3) node {\scriptsize$10$};
\draw (-2.4,2) node {\scriptsize$\phantom+10$};
\draw (-2.4,-2) node {\scriptsize$-10$};
\draw [ultra thick] (0,0) circle [radius=1];
\end{tikzpicture}
\end{center}

In the following example, \verb$trange$ is reduced
to draw a quarter circle instead of a full circle.

{\color{blue}
\begin{verbatim}
trange = (0,pi/2)
f = 5 (cos(theta),sin(theta))
draw(f,theta)
\end{verbatim}}

\begin{center}
\begin{tikzpicture}
\draw (-2,-2) -- (2,-2);
\draw (-2,0) -- (2,0);
\draw (-2,2) -- (2,2);
\draw (-2,-2) -- (-2,2);
\draw (0,-2) -- (0,2);
\draw (2,-2) -- (2,2);
\draw (-2,-2.3) node {\scriptsize$-10$};
\draw (2,-2.3) node {\scriptsize$10$};
\draw (-2.4,2) node {\scriptsize$\phantom+10$};
\draw (-2.4,-2) node {\scriptsize$-10$};
\draw [ultra thick] (1,0) arc (0:90:1);
\end{tikzpicture}
\end{center}
